# Strix Google Reviews

Плагин Google Reviews Pro для WordPress — лучший на рынке для отображения отзывов Google Places на вашем сайте.

## Особенности

- **Отображение отзывов**: Показывайте отзывы и рейтинги вашей компании
- **Виджет**: Многовариантный виджет для размещения в боковой панели
- **Шорткоды**: Гибкие шорткоды для вставки отзывов в любой контент
- **Фильтрация**: Возможность показывать только 5-звездочные отзывы
- **Кеширование**: Автоматическое кеширование для улучшения производительности
- **Адаптивный дизайн**: Отзывчивый дизайн для всех устройств
- **Кастомные отзывы**: Форма для самостоятельного написания отзывов посетителями
- **Модерация отзывов**: Система одобрения отзывов в админке
- **Демо-режим**: Показывает примеры отзывов без настройки API

## Установка

1. Скачайте плагин
2. Загрузите папку `strix-google-reviews` в директорию `/wp-content/plugins/`
3. Активируйте плагин через меню "Плагины" в WordPress
4. Перейдите в **Google Reviews** > **Settings** для настройки

## Настройка

### Получение Google Places API ключа

1. Перейдите в [Google Cloud Console](https://console.developers.google.com/)
2. Создайте новый проект или выберите существующий
3. Включите **Places API**
4. Создайте API ключ в разделе "Credentials"
5. Ограничьте API ключ для безопасности

### Получение Place ID

1. Используйте [Google Place ID Finder](https://developers.google.com/places/place-id)
2. Найдите вашу компанию
3. Скопируйте Place ID (начинается с "ChIJ")

### Настройки плагина

1. Введите ваш **Google Places API Key**
2. Введите ваш **Google Place ID**
3. Настройте параметры отображения:
   - Показывать название компании
   - Показывать веб-сайт
   - Показывать телефон
   - Показывать только 5-звездочные отзывы
   - Время кеширования

## Использование

### Шорткоды

```
[strix_google_reviews]
[strix_google_reviews place_id="ChIJd8BlQ2gVwAARRapDhqKPvCQ" limit="5"]
[strix_google_reviews show_company="0" layout="grid"]
```

**Параметры шорткода:**
- `place_id` - ID места Google (опционально, использует настройки по умолчанию)
- `limit` - Количество отзывов для отображения (по умолчанию: 5)
- `show_company` - Показывать название компании (0/1)
- `layout` - Тип макета (list/grid, по умолчанию: list)

### Виджет

1. Перейдите в **Внешний вид** > **Виджеты**
2. Найдите виджет **Google Reviews**
3. Перетащите его в нужную область
4. Настройте заголовок, Place ID и количество отзывов

### Управление отзывами

Перейдите в **Google Reviews** > **Reviews** для просмотра и обновления отзывов.

## Кастомные отзывы

### Форма отправки отзывов
```php
[strix_review_form]
```

### Отображение отзывов с формой
```php
[strix_custom_reviews]
[strix_custom_reviews limit="10" show_form="1"]
```

**Параметры:**
- `limit` - Количество отзывов (по умолчанию: 10)
- `show_form` - Показывать форму отправки (0/1, по умолчанию: из настроек)
- `pagination` - Включить пагинацию (0/1, по умолчанию: 1)

### Управление кастомными отзывами

1. **Админка** → **Custom Reviews** - управление отзывами
2. **Статусы:** Published (одобренные), Pending Review (ожидают модерации)
3. **Массовые действия:** Одобрить, отклонить, удалить отзывы

### Настройки кастомных отзывов

- **Auto-approve Reviews** - Автоматическое одобрение новых отзывов
- **Require Name** - Обязательное поле имени
- **Require Email** - Обязательное поле email
- **Show Review Form** - Показывать форму вместе со списком отзывов

## Часто задаваемые вопросы

**В: Поддерживает ли плагин шорткоды?**
О: Да, используйте `[strix_google_reviews]` для вставки отзывов.

**В: Как найти Place ID?**
О: Используйте [Google Place ID Finder](https://developers.google.com/places/place-id).

**В: Можно ли показывать отзывы разных мест на одной странице?**
О: Да, используйте разные `place_id` в шорткодах.

**В: Сколько отзывов можно отобразить?**
О: Google API возвращает до 5 наиболее релевантных отзывов.

**В: Что такое демо-режим?**
О: Демо-режим показывает примеры отзывов без настройки Google API. Полезно для тестирования внешнего вида.

**В: Как работают кастомные отзывы?**
О: Посетители могут оставлять отзывы через форму на сайте. Вы можете модерировать их в админке.
